Form Validation - HTML5

For more customized validation we can use JavaScript to validate our forms. HTML5 or JavaScript form validation should never be your sole form of form validation as it can be easily changed and or overridden by any user. Server validation of data sent to a server should always be performed in addition to HTML5 or JavaScript validation.
