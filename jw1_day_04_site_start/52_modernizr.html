Modernizr

Modernizr is a Javascript library which detects features that a browser support

You do not need to know JavaScript to use some of the features of modernizr, but knowing JavaScript is needed to utilize feature detection in your scripts.

One thing that Modernizr can do is add classes to the html tag of your web page that change depending on whether or not a browsers supports a feature.

Below is a code sample of the classes that Modernizr adds to the html tag of the latest version of Chrome and to IE8.

Modernizr can include the HTML5 shiv/shim (this adds support to IE8 and below to HTML5 tags) so you do not need to include the HTML5 shiv/shim if you are using Modenizr.

Use Modernizr to Load a Graphic for a Logo for Older Browsers

This works because Modernizr creates "no-{feature}" classes that are added to the html tag. We can then write CSS that differs depending on which class is in the html tag.

Below is sample code of two CSS style rules that target the same element but apply different styles depending on whether or not the "no-boxshadow" class has been added to the html tag or not. If the "no-boxshadow" tag was added to the html tag than Modernizr has determined that the browser does not support the css "box-shadow" property.

CSS for ".logo" if Box Shadow is Supported


/* This style will apply for browsers that support 
the box-shadow css property */
.f_20 .logo {
	font-family: Impact, Haettenschweiler, "Franklin Gothic Bold", "Arial Black", sans-serif;
	font-size: 30px;
	text-align: center;
	-wekit-box-shadow: 10px 10px 5px #888;
	-moz-box-shadow: 10px 10px 5px #888;
	box-shadow: 10px 10px 5px #888;
	width: 315px;
	height: 119px;
	padding: 0;
	line-height: 119px;
	border: 1px solid #000;
	margin: 50px auto;	
}

CSS for ".logo" if Box Shadow is Not Supported


/* Since modernizr adds a class called "no-boxshadow" to the body tag of browsers 
that do not support the box-shadow css property...this rule will run only for browsers that do not support the box-shadow css property */
.no-boxshadow .f_20 .box_21 .logo {
	background-image: url(../images/bg_awesome_designs_logo.png);
	background-repeat: no-repeat;
	height: 119px;
	padding: 0;
	border: none;
	line-height: 101px;
	text-indent: -10px;
}

Using Modernizr to conditionally load scripts

Modernizr can include the yepnope JavaScript library which allows you to conditionally load JavaScript and CSS files depending on whether or not a feature in a browser is supported.

When you build your version of Modernizr on the Modernizr download page make sure you select the Modernizr.load option.

Below we will either load a script which utilizes native CSS animations to animate a heading, or we will load a polyfill script that will use jQuery animations to animate a heading in browsers that do not support CSS animations.

Animated Heading

Code Sample using the Modnerizer.load() Method


// Use Modernizr.load method to conditionally load scripts depending
// on whether or not a browser supports a certain feature or not...

// The following code tests whether or not the browser supports
// CSS animations and loads either a script that uses CSS animations or
// a script that uses jQuery animations if the browser does not 
// support CSS animations

Modernizr.load({
  test: Modernizr.cssanimations,
  yep : 'scripts/css_animation_support.js',
  nope: 'scripts/no_css_animation_support.js'
});
